1. 이전 주에 정한 스터디 일정에 맞춰 스터디 자료 준비 및 질문 정리합니다.

4. 리뷰어에게 리뷰를 전부 받으면 merge를 진행합니다.
5. 이후 다음 스터디를 준비합니다.
